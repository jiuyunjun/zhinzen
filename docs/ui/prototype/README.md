# UI 原型 / Design Prototype

这是 [Claude Design](https://claude.ai/design) 导出的高保真可点击交互原型，作为
`apps/web` 实现的**视觉与交互参考**，不是生产代码。

直接用浏览器打开 [`Zhinzen 位置共享原型.html`](./Zhinzen%20位置共享原型.html) 即可预览
（React + Babel 在浏览器内编译运行）。

## 串通的核心闭环

1. 入场 —— 只输入显示姓名，明确「无需注册 / 登录」。
2. 创建 / 加入房间。
3. 地图主页 —— 自己（蓝点 + 朝向锥 + 精度圈）、成员图钉（在线/移动/过期/离线）、
   顶部状态栏 + 房间码 + 邀请、共享开关、回到我的位置、底部成员横滑列表。
4. 成员详情面板 —— 距离 + 相对方向小罗盘、能力标签（UWB/蓝牙/无罗盘）、
   Google 地图导航、轨迹开关、过期时提示「对方可能已移动」。
5. 全屏方向指针找人 —— 雷达 + 大箭头（对方颜色）+ 距离读数。
6. 点自己 —— 编辑显示姓名、强调色、共享开关、离开房间。

## 文件

| 文件 | 作用 |
| --- | --- |
| `Zhinzen 位置共享原型.html` | 入口，加载 React/Babel 与下列模块 |
| `core.jsx` | 设计 token、i18n（中/英）、mock 房间数据、几何工具 |
| `frames/ios-frame.jsx` | iOS 设备外壳、状态栏、液态玻璃组件 |
| `mockmap.jsx` | 程序化拟真地图 + 自己标记 + 成员图钉 + 轨迹 |
| `ui.jsx` | 图标、头像、状态点、按钮、能力标签 |
| `screens.jsx` | 入场 / 房间 / 详情 / 找人 / 自己编辑 |
| `app.jsx` | 状态机、地图页面骨架、底部成员条、Tweaks 接线 |
| `tweaks-panel.jsx` | 浮动 Tweaks 调参面板 |

> 生产实现请提取这里的**视觉产出**（尺寸、颜色、布局、交互），用目标技术栈
> （`apps/web` 的 React + TS）重建，而不是照搬原型内部结构。设计 token 已抽取到
> [`packages/shared-ui`](../../../packages/shared-ui)。
